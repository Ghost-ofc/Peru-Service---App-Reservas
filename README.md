# AppReservas

Aplicación móvil para gestión de reservas.

## Características

- Desarrollada en Kotlin y Java
- Estructura modular con Gradle
- Soporte para pruebas unitarias y de instrumentación

## Estructura del proyecto

- `app/`: Código fuente principal de la aplicación
    - `src/main/java/`: Código fuente en Java/Kotlin
    - `src/main/res/`: Recursos de la aplicación (layouts, imágenes, etc.)
    - `src/main/AndroidManifest.xml`: Manifest de la app
- `build.gradle.kts`: Configuración de Gradle
- `gradle/`: Archivos de configuración de Gradle

## Requisitos

- Android Studio 2024.3.2 o superior
- JDK 17+
- Gradle 8+

## Cómo ejecutar

1. Clona el repositorio
2. Abre el proyecto en Android Studio
3. Sincroniza Gradle
4. Ejecuta la app en un emulador o dispositivo físico
